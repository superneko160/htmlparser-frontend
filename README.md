# htmlparser

## Feature

- Parsing HTML attributes
- Download the results of parsing HTML attributes as a JSON-formatted file
- Download the results of parsing HTML attributes as a CSV-formatted file

![arch](./docs/architecture.png)

## Run

```bash
cd htmlparser
bun run dev
```

## SetUp

```bash
cd htmlparser
bun install
```

## Lint

```bash
cd htmlparser
bun lint
```

## Format

```bash
cd htmlparser
bun format
```
